# OPEX Simple 01

OPEX package with single file  

```bash
└── container
│       ├── collection
│       │   ├── asset
│       │   ├── asset.opex
│       │   └── collection.opex
│       └── container.opex

```

container can be zipped and ingested with ingest (v6)
